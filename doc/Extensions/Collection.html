  • chunk(by:)

    Creates an array of elements split into groups the length of size. If array can’t be split evenly, the final chunk will be the remaining elements. [cf.] https://lodash.com/docs/4.17.4#chunk

    Chinese description

    創建一個元素陣列，將其分為大小長度的二維陣列。 如果無法均勻分割陣列，則最後一塊將是剩餘的元素。

    Use example

       ['a', 'b', 'c', 'd'].chunk(by: 2)
   // [['a', 'b'], ['c', 'd']]

   ['a', 'b', 'c', 'd'].chunk(by: 3)
   // [['a', 'b', 'c'], ['d']]

    Declaration

    Swift

    func chunk(by size: Int) -> [[Element]]?

    Parameters

    size

    The length of each chunk

    Return Value

    SafeCollectionable

  • group(by:)

    Creates a dictionary with key is hash. (like SQL: GROUP BY)

    Chinese description

    創建由雜湊鍵拆分的字典。

    Use example

       [1585701811, 1585788211, 1585961011].group {
       return Double($0).dateSince1970.toString(format: "yyyy/MM/dd")
   }
   // ["2020/04/01": 1585701811, "2020/04/02": 1585788211, "2020/04/04": 1585961011]
