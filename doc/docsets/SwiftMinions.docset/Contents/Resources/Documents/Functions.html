  • ifLet(value:success:fail:)

    A convenience syntax for both success and fail unwrapped

    Chinese description

    一個方便的語法糖，讓你可以處理 unwrapped 成功跟失敗的時候

    Use example

    let c: String? = "Ya"
let b = ifLet(value: c,
              success: { value in "Success" + value },
              fail: { "Oh noooo!" })

    Declaration

    Swift

    public func ifLet<T, U>(value: T?,
                 success thenFunc: (T) -> (U),
                 fail elseFunc: () -> (U)) -> U
